# URL Shortener (TinyURL Clone)

A simple URL shortening web application built using Python and Flask.

## Features

- Convert long URLs into short shareable links
- Redirect short URLs to the original links
- Store URL mappings using SQLite
- Simple web interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML

## Project Structure

url-shortener
│
├── app.py
├── urls.db
└── templates
    └── index.html

## How to Run

1. Install Flask

pip install flask

2. Run the application

python app.py

3. Open browser

http://127.0.0.1:5000
